Hierarchical

  1. Split the total one-sided \(\alpha = 0.025\) evenly between the two families
    • C-family: Arms C high-dose and C low-dose
    • C*-family: Arms C* high-dose and C* low-dose
  2. Within each family
    1. If the p-value from the high-dose arm is \(< 0.0125\), reject the null hypothesis for the high-dose arm. Continue to the next step.
      • Otherwise, fail to reject the null hypothesis for both the high- and low-dose arm. Stop.
    2. If the p-value from the low-dose arm is \(< 0.0125\), reject the null hypothesis for the low-dose arm.

Hochberg

  1. Split the total one-sided \(\alpha = 0.025\) evenly between the two families
    • C-family: Arms C high-dose and C low-dose
    • C*-family: Arms C* high-dose and C* low-dose
  2. Within each family
    1. Sort the two p-values
    2. If the smaller p-value is \(< 0.00625\), reject the null hypothesis for corresponding arm. Continue to the next step.
      • Otherwise, fail to reject the null hypothesis for both the high- and low-dose arm. Stop.
    3. If the larger p-value is \(< 0.0125\), reject the null hypothesis for the corresponding arm.
